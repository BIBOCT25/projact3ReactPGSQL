Шаг 1 — Хук useMobile

Отслеживание размера экрана

Идея: в React нет встроенного способа использовать CSS media-query в логике. Хук useMobile подписывается на событие resize окна, обновляет состояние и возвращает isMobile: boolean. Компонент перерисовывается автоматически при смене ширины.
1

Хук useMobile — добавить в App.tsx

Хук встроен прямо в файл App.tsx (не выносим в отдельный файл — используется только в Header).

📄 client/src/App.tsx
// Лаб 15: хук для отслеживания мобильного экрана
function useMobile(breakpoint = 768) {
  const [isMobile, setIsMobile] = useState(window.innerWidth < breakpoint)

  useEffect(() => {
    const handler = () => setIsMobile(window.innerWidth < breakpoint)
    window.addEventListener('resize', handler)
    return () => window.removeEventListener('resize', handler) // cleanup
  }, [breakpoint])

  return isMobile
}
Cleanup важен: без removeEventListener при размонтировании компонента обработчик останется в памяти и продолжит вызываться — утечка памяти.

Адаптивная шапка с drawer-анимацией

Структура состояния и загрузка

export function FavoritesPage() {
  const { user } = useAuth()
  const navigate = useNavigate()
  const { addToCart } = useCart()
  const { toasts, show, dismiss } = useToast()

  const [favs, setFavs]       = useState<any[]>([])
  const [loading, setLoading] = useState(true)
  const [removing, setRemoving] = useState<Set<number>>(new Set())

  useEffect(() => {
    if (!user) { navigate('/auth'); return } // guard для гостей
    api.getFavorites()
      .then(data => setFavs(data))
      .catch(() => show('Ошибка загрузки', 'error'))
      .finally(() => setLoading(false))
  }, [user])
}

Удаление одного товара (оптимистичное UI)

Сначала помечаем как «удаляемый» (opacity 0.5), вызываем API, убираем из массива. При ошибке показываем toast.

const remove = async (productId: number) => {
  setRemoving(prev => new Set([...prev, productId])) // fade
  try {
    await api.toggleFavorite(productId)
    setFavs(prev => prev.filter(f => f.productId !== productId))
    show('Удалено из избранного')
  } catch {
    show('Не удалось удалить', 'error')
  } finally {
    setRemoving(prev => { const s = new Set(prev); s.delete(productId); return s })
  }
}
